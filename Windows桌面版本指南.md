# Windows桌面版本指南

本指南将帮助你在Windows电脑上直接运行立体几何学习应用，无需Android模拟器或手机。

## 整体优势
- ✅ **无需安装Android Studio或模拟器**
- ✅ **原生Windows性能**（启动快，运行流畅）
- ✅ **直接双击运行**（像普通Windows软件一样）
- ✅ **离线可用**（下载后无需网络）

## 快速开始（3步）

### 第一步：触发GitHub构建
1. **访问你的仓库Actions页面**：https://github.com/luoyabo/Geometry3DApp/actions
2. **找到"Build Windows Executable"工作流**
3. **点击"Run workflow" → "Run workflow"**（手动触发）

### 第二步：等待构建完成（10-15分钟）
- 首次构建较慢（需下载Flutter和Windows SDK）
- 可实时查看构建日志
- 所有步骤显示绿色对勾✓表示成功

### 第三步：下载并运行
构建完成后：
1. **下载EXE文件**：在"Artifacts"区域下载：
   - `windows-executable` → `geometry_3d_app.exe`（主程序）
   - `windows-release-zip` → 完整发布包（推荐）
2. **解压ZIP文件**（如下载的是ZIP包）
3. **双击运行**：`geometry_3d_app.exe`

## 详细步骤

### 1. 构建状态检查
访问：https://github.com/luoyabo/Geometry3DApp/actions

如果看到：
- ✅ **绿色对勾**：构建成功，跳到下载步骤
- 🟡 **黄色圆圈**：正在构建，等待完成
- ❌ **红色叉叉**：构建失败，查看错误日志

### 2. 下载可执行文件
**推荐下载**：`windows-release-zip`（包含所有依赖文件）

**文件说明**：
- `geometry_3d_app.exe`：主程序（约50-80MB）
- `flutter_windows.dll`：Flutter运行时库
- `flutter_assets/`：应用资源文件
- `data/`：应用数据目录

### 3. 运行应用
#### 方法A：直接双击（最简单）
1. 找到下载的`geometry_3d_app.exe`
2. 双击运行
3. 如提示"Windows保护了你的电脑"，点击"更多信息" → "仍要运行"

#### 方法B：创建桌面快捷方式
1. 右键点击`geometry_3d_app.exe`
2. 选择"发送到" → "桌面快捷方式"
3. 双击桌面图标运行

#### 方法C：命令行运行
```cmd
cd 解压目录
geometry_3d_app.exe
```

### 4. 应用功能验证
运行后检查：
1. **主菜单**：点击各个功能卡片
2. **3D形状**：鼠标拖动旋转，滚轮缩放
3. **线面角**：滑块控制角度演示
4. **三垂线定理**：查看定理图解
5. **知识点**：滚动查看内容

## 系统要求

### 最低配置
- **操作系统**：Windows 10（64位）或更高
- **内存**：4GB RAM
- **存储空间**：200MB可用空间
- **图形**：支持DirectX 9及以上

### 推荐配置
- **操作系统**：Windows 11（64位）
- **内存**：8GB RAM或更高
- **存储空间**：500MB可用空间
- **图形**：集成显卡或独立显卡

## 常见问题解决

### 1. 无法运行（缺少DLL）
**错误**：`无法启动此程序，因为计算机中丢失VCRUNTIME140.dll`

**解决方案**：
1. 安装Visual C++ Redistributable：
   - 下载：https://aka.ms/vs/17/release/vc_redist.x64.exe
   - 运行安装程序，重启电脑

2. 或下载完整ZIP包（包含所有依赖）

### 2. 安全警告
**警告**：`Windows已保护你的电脑`

**解决方案**：
1. 点击"更多信息"
2. 点击"仍要运行"
3. （可选）添加排除项：
   ```
   设置 → 隐私和安全性 → Windows安全中心 → 病毒和威胁防护 → 病毒和威胁防护设置 → 管理设置 → 排除项 → 添加排除项 → 选择文件/文件夹
   ```

### 3. 应用闪退
**可能原因**：
- 系统不兼容（需Windows 10+）
- 缺少运行库
- 杀毒软件拦截

**解决方案**：
1. 查看事件查看器错误日志
2. 以管理员身份运行
3. 暂时关闭杀毒软件

### 4. 构建失败
**常见错误**：
- **依赖包不兼容**：`flutter_3d_view`可能不支持Windows
- **Flutter版本问题**：需要Flutter 3.0+支持Windows
- **内存不足**：GitHub Actions资源限制

**解决方案**：
1. 查看构建日志具体错误
2. 联系开发者调整依赖
3. 重试构建

## 高级选项

### 1. 创建安装程序
如需创建专业的安装程序：
1. 使用Inno Setup：http://www.jrsoftware.org/isinfo.php
2. 创建安装脚本
3. 生成`Setup.exe`，包含开始菜单项和卸载程序

### 2. 自定义图标
替换应用图标：
1. 准备ICO文件（256×256，多尺寸）
2. 替换`windows/runner/resources.rc`中的图标引用
3. 重新构建

### 3. 添加自动更新
使用第三方库实现自动更新：
- `flutter_autoupdate`
- 或自定义更新检查器

## 与Android版本对比

| 特性 | Windows桌面版 | Android手机版 |
|------|---------------|---------------|
| **运行环境** | Windows 10/11直接运行 | 需要安卓手机或模拟器 |
| **性能** | 原生性能，启动快 | 依赖手机性能 |
| **交互** | 鼠标+键盘，精准控制 | 触摸屏手势 |
| **部署** | EXE文件，双击运行 | APK安装，需要权限 |
| **更新** | 手动替换EXE | 通过应用商店或APK |
